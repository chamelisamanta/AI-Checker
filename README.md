# AI Checker
AI Checker is a simple browser-based web app that analyzes pasted text and estimates whether it contains patterns commonly found in AI-generated writing. It also includes a revision studio that helps rewrite formal text into a more natural, human-sounding draft.

## Features
- Paste and analyze any text
- AI-likelihood percentage score
- Word count and sentence count
- Average sentence length analysis
- Vocabulary repetition percentage
- Clear signal explanations
- Rule-based rewrite tool
- Copy rewritten draft
- Responsive design for desktop and mobile
- No login, backend, API key, or database required

## Tech Stack
- HTML
- CSS
- JavaScript
